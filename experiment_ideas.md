# AI 설문 응답을 위한 페르소나 구축 및 답변 품질 측정 방안  

1. 실험 대상 페르소나 구축 (3단계: 1. 기본 LLM, 2. 기본 페르소나 3. 확장 페르소나)  
    1-1. 기본 LLM: 별도 튜닝 없는 기본 LLM; 성능 Baseline  
    1-2. 기본 페르소나: 나이, 성별, 거주지, 국적, 혼인 여부, 교육 수준, 직업, 종교, 가치관, 관심사, 등  
    1-3. 확장 페르소나: 유년기 경험, 삶에서 가장 안좋은 경험 등 -> 구체적 경험 추가; 하나씩 도입하며 단계별로 비교  

2. 설문지 작성  
    2-1. 설문 배경: 직업 만족도 조사  
    2-2. 설문 내용: 나이, 성별, 등 인구통계 항목; 취미, 관심사, 등 개인 특성 관련 항목; 직업, 직무, 근로시간, 이직 의사 등 직업 관련 항목  

3. 설문 결과 분석  
    3-1. 교차 분석: 분산분석을 통한 그룹별(성별, 연령, 등) 만족도 차이 분석  
    3-2. 회귀 분석: 회귀분석을 통한 만족도 영향 요인 분석  

4. 품질 기준 정의  
    4-1. AI 성능 측정 기준:  
        - 정의된 페르소나와 답변의 일관성  
        - AI 인터뷰(?)  
        - 답변 다양성 (분산도) -> 표준편차, 엔트로피  
    
    4-2. 설문 답변 품질 측정 기준:  
        - 답변 다양성 (분산도) -> 문항별 표준편차, 엔트로피  
        - 반복 선택 (복수 선택 질문에서 거의 모든 항목 선택, 같은 항목 반복 선택) -> 응답자별 표준편차, 엔트로피  
        - 상충 응답 빈도 (응답의 일관성) -> 연관된 문항 간 상관 관계  
        - 매우 만족/매우 불만족 응답 비율 -> ceiling/floor effect, 평균값, 응답자 편향 or 왜곡된 문항  
        - 표본 품질 평가 -> 성, 연령, 산업 분포 대표성, Chi-square test  

        - 타당도 평가 -> EFA/CFA (On Hold, 요인 추출을 위한 지표)  
        - 신뢰도 평가 -> Cronbach's alpha (On Hold, 요인 추출을 위한 지표)  

5. 개선 방향 실험  
    5-1. 페르소나 확장 기반 실험: 1-3 참조  
    5-2. Modelfile 파라미터 기반 실험: 파라미터 변경 후의 설문 답변 품질 변화 측정  
    5-3. AB 테스트 기반 품질 검증: 5-1 및 5-2로 인한 변화에 따른 답변 품질 변화 AB 테스트 기반 통계적 검증  







### 설문 응답 패턴 -> 특정 성격, 직업, 등을 가진 사람은 특정한 응답 패턴을 갖는다.  
### 타겟층에 변화를 주면 (특정 속성을 갖춘 페르소나만 이용하면) 만족도에 변화가 발생하는가?  
    -> 디테일한 비즈니스 인사이트 제공 가능  
    -> Ex. 현재 구축한 서비스가 타겟층을 제대로 겨냥하는지, 혹은 예상치 못한 다른 타겟층(수요)이 존재하는지?  
    -> 실제 사람과 유사한 AI 페르소나 기반 설문 시뮬레이션 가능
